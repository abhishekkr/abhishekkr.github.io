<b>Brief Description:
</b>
</div>
<div>One of the best network analysis utilities on Windows.&nbsp;
</div>
<div>You can choose any Physical/Virtual NIC present on your machine to capture traffic.
</div>
<div>It shows the traffic in a nested tree menu ordered by Application Name generating that traffic.
</div>
<div>This way you can analyse entire network traffic per application from your machine.
</div>
<div>Tool to allow capturing and protocol analysis of network traffic.
</div>
<div>It's free, it's fine network packets analysis utility.
</div>
<div>

</div>
<div>It also allows you to save captured data into PCAP files, and can even open and analyze PCAP files.
</div>
