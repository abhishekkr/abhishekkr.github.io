<b>[] A try to check if Machine is infected by a Kneber (Zeus Variant)
</b>, is

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; The registry key can be found by following this path, he said:

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; HKLM\Software\Microsoft\Windows NT\CurrentVersion\Winlogon\UserInit

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; normally will have an entry like "C:\WINDOWS\system32\userinit.exe,"
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ZeuS will add itself to the list, typically as 'ntos.'
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; But could always change its name; so if any un-relevant entries found here... may be machine is infected.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; If any more entries found, or suspicion is there scan the file listed here.
