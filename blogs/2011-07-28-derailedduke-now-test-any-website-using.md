<span class="Apple-style-span" style="font-family: Georgia,'Times New Roman',serif;">" to install all required gems.

<span class="Apple-style-span" style="font-family: Georgia,'Times New Roman',serif;">
<b>Step#3&nbsp;

<span class="Apple-style-span" style="font-family: Georgia,'Times New Roman',serif;">If your ruby binary file is accessible by any name other than 'ruby' {i.e. say you are jruby and thus access it using '

<span class="Apple-style-span" style="font-family: Georgia,'Times New Roman',serif;">" and replace the value of variable "

<span class="Apple-style-span" style="font-family: 'Courier New',Courier,monospace;">RUBY_BIN
